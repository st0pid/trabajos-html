Lista de compra para PC

Componentes del PC (desordenada)

  • Procesador (CPU)
  • Tarjeta gráfica (GPU)
  • Placa base (Motherboard)
  • Memoria RAM
  • Almacenamiento (SSD/HDD)
  • Fuente de alimentación (PSU)
  • Gabinete
  • Monitor
  • Teclado y ratón

Pasos para ensamblar un PC (ordenada)

  1. Coloca el CPU en la placa madre
  2. Instalar la memoria RAM
  3. Montar la placa base en la caja
  4. Conectar la fuente de alimentación
  5. Instalar la tarjeta gráfica
  6. Conecta todo al almacenamiento
  7. Verificar todas las conexiones
  8. Encender y probar el sistema
